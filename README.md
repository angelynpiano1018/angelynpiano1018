### 🌟 Inventory Management System
A comprehensive Java-based inventory management application developed in **Computer Programming 2 (CompProg 2)** using NetBeans IDE.

**Project Overview:**
This system provides a complete solution for managing inventory operations with the following capabilities:
- **Product Management**: Create, read, update, and delete product records
- **Stock Management**: Track inventory levels and monitor stock availability
- **Inventory Records**: Maintain detailed logs of all inventory transactions and movements
- **User-Friendly Interface**: Intuitive GUI built with Java Swing for seamless navigation

**Technical Stack:**
- Language: Java
- IDE: NetBeans
- Database: MySQL
- Design Pattern: MVC (Model-View-Controller)

**Key Learnings:**
- Database design and normalization
- GUI development with Java Swing
- CRUD operations and data persistence
- Project management and documentation
